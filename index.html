SuperSender

USER GUIDE


Overview

SuperSender compiles digital mailing lists and creates email templates for the user.

Workflow:

  1. Add companies and people to your project.
  2. Assign positions to people.
  3. Build your lists.
  4. Create your email templates.

Important:
List compilation is based on company & position. A person is included in a list only if BOTH of the following are true:

  • The person's company is included on the list.
  • The person is assigned a position that is included on the list.

If either condition is not met, the person will be omitted from the list.


Step-by-Step

Companies

  1. Add companies, assigning each a name.
  2. A unique Company_ID is automatically assigned.

People

  1. Add people, assigning:
    • An email address
    • A company (if left blank, they will be marked as "UNASSIGNED (id: 0)")
  2. A unique Person_ID is automatically assigned.
  3. Create positions (groups & points) by adding columns.
  4. Select applicable positions for each person.

Lists

  1. Create lists by assigning a name (List_ID is automatically assigned).
  2. Select which positions from which companies are to be added to the list.
    • Inclusive = ALL companies included EXCEPT for those listed as Exceptions
    • Inclusive = NO companies included EXCEPT for those listed as Exceptions
  3. You can:
    • Copy the email addresses to clipboard.
    • Build an email template for commonly distributed emails.

Emails

  1. Create emails and assign each a name (Email_ID is automatically assigned).
  2. Select which lists go in the To, Cc, and Bcc fields.
  3. Edit all other fields as necessary.
  4. Click Preview to inspect the email and ensure it's error-free.
  5. Click Download to export the email draft as a .EML file.
  6. Drag & drop the .EML file into your Email Client Application's Drafts folder.
  7. Review and send.

Data Storage

SuperSender does not require sign-in or store any table data on an external database. All data:

  • Is stored locally in your browser during use (via Local Storage).
  • Can be downloaded as an Excel file.
  • Can be re-uploaded to restore your session later.

Note: Local Storage persistence depends on your browser and its configuration. To avoid data loss, always download and save your Excel file after making changes.

Tools Available:

  • Recover Memory - Restore session data from Local Storage.
  • Clear Memory - Permanently erase session data from the browser.

Editing from Saved Excel File

Some actions cannot be done directly in the SuperSender UI, including:

  • Duplicating a company/person/list/email.
  • Changing an ID# (Company_ID, Person_ID, etc.).
  • Duplicating or renaming a position (column) on the People table.
  • Adding new informational columns to the Company table.

To perform these actions, manual changes to the excel file is required:

  1. Go to the File tab and click Save to File.
  2. Open the downloaded Excel file and make your changes.
  3. Use Upload from File to import your updated data.

Caution: Do not:

  • Edit any columns or column headings in the List or Email tables.
  • Edit any ID column headings.
  • Allow any repeat ID numbers.

Data Type Symbols

In the Excel file, column names include symbols indicating their data types:

SymbolData Type
!Boolean - Point
#Boolean - Group
%Array of Integers
*Array of Strings
&Integer
_String (also applies if no symbol is present)

Points & Groups

Currently, groups and points function the same. In future versions, points will be distinct positions that only one person per company can hold (e.g., Point Man for XYZ).


Uploading

Caution: When you upload a file using Upload from File, all current table data in your open page will be permanently overwritten.

Before uploading:

  • Ensure your Excel file was originally downloaded using the Save to File feature, or closely follows that template.
  • Only edit the Excel file using the guidelines provided in the "Editing from Saved Excel File" section.
  • Avoid modifying any ID columns or the column structure of the List or Email tables.

Downloading / Saving

To back up your current data or prepare for manual editing, you can use the Save to File option to download an Excel file containing all your current tables and configurations.

This file can be:

  • Re-uploaded later to restore your session.
  • Edited outside of SuperSender (with care and following the proper guidelines).
  • Used as a template to convert an existing contact list into the correct format.

Tip: Regularly download your project file after making changes to avoid losing progress or relying solely on browser storage.
